# Goal-Management Application

## Overview
Goal-Management is a student-oriented application designed to help manage medium to long-term goals. It integrates the management of assignments, qualification exams, planned study sessions, and daily life habits. The app also features a lightweight social networking function that allows users with similar goals to connect and maintain motivation.

## Features

### 1. Dashboard
- Today's Tasks
- Summary of progress on medium to long-term goals
- Visualization of health data (sleep, diet, mood)
- Weekly and monthly performance tracking

### 2. Goal Setting
- Long-term, medium-term, and short-term goals
- Deadlines, categories, and priorities
- Achievement metrics (percentage, count, custom indicators)

### 3. Task Management
- Add tasks for assignments and exam preparation
- Scheduling and deadlines
- Progress updates and completion tracking

### 4. Health and Lifestyle Management
- Sleep tracking
- Simple food logging
- Mood index
- Automatic performance analysis

### 5. Journal and Reflection
- Record entries in Markdown format
- Daily summaries
- Historical reflections (weekly/monthly)

### 6. Social Features (Lightweight)
- Display users with similar goal categories
- Simple commenting and liking functionality
- Profile editing

## Getting Started

### Prerequisites
- Node.js (version 14 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```
   git clone <repository-url>
   cd goal-management-app
   ```

2. Install dependencies:
   ```
   npm install
   ```

### Running the Application
To start the development server, run:
```
npm run dev
```
Open your browser and navigate to `http://localhost:3000` to view the application.

### Building for Production
To create a production build, run:
```
npm run build
```
