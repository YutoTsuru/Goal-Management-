# Goal-Management-
# 学生向け目標管理アプリ
課題や資格試験に向けてのタスクを管理、ユーザーが目標に対する予定管理を楽に行うことができるツール<br>

授業の一環で作成<br>

【要件】<br>
中長期目標を管理するアプリ。簡易的なSNS機能があり、似た目標を持った人と交流でき、モチベーションに管理にもつながる。【機能例】ダッシュボード、目標設定、進捗管理、体調管理、食事や睡眠などからパフォーマンス分析、日誌、振り返る<br>
